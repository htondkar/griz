A grid system for React applications based on [styled-components](https://www.styled-components.com).

### Usage
```jsx
import {Grid, GridCol} from 'griz';

const App = () => (
  <Grid>
    <GridCol>Hi<GridCol>
    <GridCol>Hi<GridCol>
  </Grid>
);
```
